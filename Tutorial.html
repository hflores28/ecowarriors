Como usar Git:

  1. Copiar el proyecto a nuestro entorno local:

    2. git clone (url)

  2. Traer lo que esta en el repositorio

    3. git pull

  3. Ver estado del proyecto

    4. git status

  4. Guardar cambios en archivos

    5. git add . //Agrega todos los archivos git add (nombre_archivo)

  5. Subir cambios "mensaje"

    6. git commit -m "mensaje_commit"

  6. Subir cambios "Repositorio"

    7. git push origin master

Crear proyecto en Unity

  1. Abre Unity Hub
  2. Crear un nuevo proyecto con plantilla 2D o 3D
  3. Nombrar el proyecto "EcoWarriors"

Configurar el proyecto

  1. Crear un nuevo proyecto con plantilla 2D o 3D
  2. Nombrar el proyecto "EcoWarriors"
  3. En 'Edit > Project Settings' ajustamos la Resolución y ajustes de graficos

Organización de carpetas

    Assets/
    ├── Art/
    ├── Audio/
    ├── Prefabs/
    ├── Scenes/
    ├── Scripts/
    ├── UI/

Importar recursos

  1. Importar graficos usando Aseprite y Photoshop 'Assets/Art'
  2. Importar audio usando Logic Pro y Pro Tools 'Assets/Audio'

Crear las escenas

  1. En 'File > New Scene' Guardar en 'Assets/Scenes'
  2. Configurar la escena elementos y objetos para el primer nivel

Desarrollo de Scripts de Movimiento

  1. Para gestionar movimientos creamos un Script en 'Assets/Scripts'
  2. Se crearan scrips adicionales para cada habilidad de los personajes "Luna, Kai, Aria, Orion"

Diseño de Niveles

  1. Nivel 1: Limpieza de Playas
    2. Añadir Escena, Objetos y NPCs que eduquen sobre el reciclaje
  2. Niveles adicionales (level 2, level 3)
    3. Por cada nivel del juego Diseñar cada nivel de acuerdo a los temas (reforestación, energías renovables)

Configuración de Controladores

  1. Asignación de teclas de movimiento y habilidades
  2. Configuración de controles PC-Movil

Integración y pruebas

  1. Asignación de teclas de movimiento y habilidades
  2. Configuración de controles PC-Movil

Configuración Inicial

MainMenu: Crear un Canvas con botones para "Start Game" y "Quit Game". Añadir el script MainMenuUI al Canvas y asignar las funciones a los botones.
HUD: Crear un Canvas con elementos de texto para mostrar la XP y Eco Coins.
Añadir el script HUD al Canvas y asignar los elementos de texto a las variables públicas del script.
Player: Crear un prefab del jugador con el componente PlayerController.
NPCs e Ítems: Crear prefabs para NPCs e ítems con los respectivos scripts NPC e Item.
Level Manager: Añadir el script LevelManager a un objeto vacío en cada nivel para gestionar la carga y recarga de niveles.
Game Manager: Añadir el script GameManager a un objeto vacío en la escena principal y asegurarse de que persista entre escenas utilizando DontDestroyOnLoad.

Recursos

  1. Art: Coloca todos los recursos gráficos (sprites, tilesets) en la carpeta Art/.
  2. Audio: Coloca todos los archivos de audio (música, efectos de sonido) en la carpeta Audio/.
  3. Prefabs: Guarda todos los prefabs creados (jugadores, NPCs, ítems) en la carpeta Prefabs/.
  4. UI: Guarda todos los elementos y scripts relacionados con la UI en la carpeta UI/.
